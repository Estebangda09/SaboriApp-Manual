Preguntas Frecuentes y Soporte Técnico

Respuestas operativas rápidas para resolver inconvenientes en sucursal de forma inmediata.

Verifica haber guardado los cambios en la sucursal correspondiente y que la pantalla de TV tenga conexión a internet activa para recibir la señal en tiempo real.

Sí, SaboriApp permite mapear diferentes pantallas inteligentes por sucursal habilitando categorías individuales y diseños independientes según la infraestructura del local.

No se necesita hardware costoso o de última generación. SaboriApp funciona a través de un navegador web estándar. Cualquier televisor Smart TV convencional o dispositivo intermedio de transmisión (como TV Box o Chromecast) con conectividad web puede cargar la URL del salón de ventas.

¡Es muy sencillo! Una vez que abras el enlace de tu pantalla, con hacer un solo clic con el mouse en cualquier parte del monitor (o presionar cualquier tecla) el sistema activará automáticamente el "Modo Pantalla Completa". Para salir, presiona la tecla Esc en tu teclado.
